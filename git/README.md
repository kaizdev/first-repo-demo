## Commands

### pwd
- prints the working directory
- code snippets using ```
```bash
    pwd
```


```js
    const myVar = 'hello'

```


```html
    <h1>Test</h1>
    <p>Hello</p>
```

### cd
- changes into specified directory

```bash
    cd Code
```


### ls
- Lists files in current directory
```bash
    ls
```

### mkdir
- Creates a new directory
```bash
    mkdir examples
```

### touch
- creates a new file in the current directory
```bash
    touch index.html
```
