Walkthrough for making changes to a repo through a pull request

Steps to take

  1. Create Repo on GitHub
  2. git init
  3. git remote add origin [link to repo]
  4. git add .
  5. git commit -m 'what updates did we do'
  6. git push origin main

General advice

  • Remote is only required at the start - not required each time. Steps 3 - 6 is required each time
  • Create a branch using git checkout -b "name of branch"
  • Files in red have cahnges made but not yet added. Staged files are in green
