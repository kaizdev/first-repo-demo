Imperative Programming

  • Imperative Programming is where we provide the computer with step-by-step instructions on how to perform a particular task.
  • Java, PHP, Javascript are examples that focus on imperative paradigm

Declarative Programming

  • Declarative Programming involves specifying what result we're expecting from our code.
  • SQL is an example which focuses on declarative paradigm
