# Forms



## Why do we need forms?
- To perform most applications in an application
    - Register
    - Login
    - Create data
    - Update data
    - Search / filter

## How do we create forms

```html
<form>
    <!-- form children go here -->

</form>
```

## What kind of form input can we use?
- text
- button
- checkbox
- radio button
- submit
- dropdown / select
- textarea
- email
- telephone
- password
- number
